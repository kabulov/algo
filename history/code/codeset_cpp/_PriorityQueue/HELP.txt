
		------PriorityQueue HELP------


Среда разработки : MS Visual Studio 2012

Цель : Реализация _приоритетной кучи_

Состоит из 3 файлов :

-PriorityQueue.hpp :  реализация приоритетной кучи
-Test.hpp : 	тестовый класс, с методами-тестами, проверяющими кучу на случайных и неслучайных тестах
-main.cpp : 	запуск кучи на тестах или пользовательском вводе

---main.cpp---

-main() - точка входа
-testAll() - запуск всех предопределенных тестов (TEST_MODE)
-RunManual() - работа с пользовательским вводом
-usage() - подсказка

---Test.hpp----

-RunAllTests() - запуск всех предусмотренных тестов
-SmallTest() - тесты небольшого размера
-RandomTest() - случайные тесты (случайно все : последовательность запросов, вставляемые в кучу числа)
-SingleValueTest() - тесты, где все вставляемые в кучу числа равны
-AlternatingQueriesTest() - равномерные участки из запросов одинакового типа
-TestOneCase() - сравнение встроенной кучи (std::priority_queue) и реализованной PriorityQueue на одном конкретном тесте

---PriorityQueue.hpp---

названия методов говорят за себя и 
совпадают с названиями соответствующих методов, определенных в std::priority_queue
