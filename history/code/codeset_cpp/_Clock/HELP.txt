

		------------Clock HELP------------

Среда разработки : MS Visual Studio 2012

Цель : 
 Вычисление угла между часовой и минутной стрелками часов.  
 Момент времени вводится из консоли.
 Поддержка трех форматов вывода угла.

Формат ввода :
 TIME FORMAT_TYPE CLOCK_TYPE 
  где

 TIME : (формат задаваемого времени)
-HH:MM --- 0 <= HH < 24, 0 <= MM < 60
-HH:MM (AM|PM) --- 0 <= HH < 12, 0 <= MM < 60
	
 FORMAT_TYPE : (формат вывода)
-dms --- вывод будет выглядеть так : d.mm'ss'' 
  Degrees Minutes Seconds, где d-градусы, mm-минуты, ss-секунды. 60mm = 1d, 60ss = 1mm (все числа - целые)
-deg --- на выходе - угол в градусах (целое число)
-rad --- на выходе - угол в радианах (целое число)

 CLOCK_TYPE : (тип часов)
-mech - механические часы : положение часовой стрелки зависит от положения минутной стрелки
 (каждый очередной переход минутной стрелки на следующую минуту поворачивает часовую стрелку на 0.5 градуса)
-quar - кварцевые часы : положение часовой стрелки не зависит от положения минутной стрелки

Комментарий к решению : 
 при данных ограничениях (нет зависимости часовой и минутной стрелок от секундной)
 угол между стрелками (часовой и минутной) либо целый (в градусах) (для кварцевых) , 
 либо имеет дробную часть 1/2 (0.5) (для механических). Причем 1/2 соответствует в формате dms : d.30'00''
 Поэтому принято решение хранить угол во внутреннем представлении соответствующих классов 
 в виде : (градусы, минуты - оба целые) т.к. этого достаточно для хранения всей информации про угол.

Все в одном файле :
---Clock.cpp---

Содержимое Clock.cpp (в порядке определенном в файле) :

-usage() - подсказка пользователю
-SplitString() - разбиение строки по шаблону
-Time - класс, представляющий время
-TimeBuilder - фабричный метод, создает объект Time, если параметр удовлетворяет правильному формату времени, 
  иначе кидает исключение. 
-Angle - класс инкапсулирующий угол (и формы его вывода)
-AngleToString - вывод Angle в запрошенном виде (формате), кидает исключение, если неправильный формат
-Clock - родительский класс, инкапсулирующий часы, имеет чисто виртуальный метод, реализуемый в потомках : 
  механическом(MechanicClock) и кварцевом (QuartzClock) часах
-MechanicClock - конкретная реализация часов, механических
-QuartzClock - конкретная реализация часов, кварцевых
-ClockBuilder - фабрика, строит(возвращает) часы в зависимости от параметра, кидает исключение, если параметр неправильный
-ProcessCMD() - работа с консолью, запуск решения(программы) на пользовательском вводе
-RunAllUnitTests() - запуск всех предопределенных тестов (TEST_MODE)
-main() - точка входа
-BuildTimeTest() - проверка TimeBuilder::BuildTime()
-MechanicClockTest() - проверка MechanicClock::AngleBetweenArrows()
-QuartzClockTest() - проверка QuartzClock::AngleBetweenArrows()
-REQUIRE_EQUAL - проверка равенства двух объектов
-time_to_string() - перевод пары {часы, минуты} в формат HH:MM
-ClockType - тип часов (механический кварцевый)
-AngleBetweenArrowsDegrees() - тестовый, правильный, вычислитель угла между стрелками, только градусы (целая часть, без дробной)
-AngleBetweenArrowsMinutes() - тестовый, правильный, вычислитель угла между стрелками, только минуты (дробная часть градусов)
-ClockTest() - проверка AngleBetweenArrows
